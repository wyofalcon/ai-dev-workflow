## Follow-up Task: EasyCv Overlay Architecture

### Problem

When the Easy CV chat is minimized, it shows a black screen because the wizard is rendered as a separate route (/easy-cv).

### Solution

Convert EasyCvWizard to work as an **overlay** that renders on top of the landing page.

### Requirements

#### 1. Render LandingPage as background

- Import LandingPage into EasyCvWizard
- Render LandingPage behind the modal/drawer
- When minimized, the landing page is fully visible and interactive
- When expanded, the chat overlay is on top with a backdrop

#### 2. State sync

- Minimized state should show full landing page (user can interact with it)
- Expanded state shows the chat popup overlaying the landing page
- Add a subtle backdrop when chat is open

#### 3. Minimized FAB button

- When minimized, show a floating action button (FAB) to re-open the chat
- Position bottom-right corner
- Use the sparkle/AI icon
- Add a subtle animation or pulse effect

### Implementation

In EasyCvWizard.js:

```jsx
import LandingPage from "./LandingPage.js";
import Fab from "@mui/material/Fab";

// Add state
const [isMinimized, setIsMinimized] = useState(false);

// In render:
return (
  <>
    {/* Landing page always renders behind */}
    <LandingPage />

    {/* Backdrop when chat is open */}
    {!isMinimized && (
      <Box
        sx={{
          position: "fixed",
          top: 0,
          left: 0,
          right: 0,
          bottom: 0,
          bgcolor: "rgba(0,0,0,0.5)",
          zIndex: 1200,
        }}
      />
    )}

    {/* Chat popup - existing modal */}
    {!isMinimized && (
      <Box
        sx={{
          position: "fixed",
          zIndex: 1201,
          // ... existing modal positioning
        }}
      >
        {/* Existing modal content */}
      </Box>
    )}

    {/* Minimized FAB button */}
    {isMinimized && (
      <Fab
        color="secondary"
        onClick={() => setIsMinimized(false)}
        sx={{
          position: "fixed",
          bottom: 24,
          right: 24,
          zIndex: 1200,
          animation: "pulse 2s infinite",
        }}
      >
        <AiIcon />
      </Fab>
    )}
  </>
);
```
