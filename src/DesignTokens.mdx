## Spacing

All spacing uses an 8px base unit:

| Token | Value | Usage           |
| ----- | ----- | --------------- |
| xs    | 4px   | Tight spacing   |
| sm    | 8px   | Default gap     |
| md    | 16px  | Section padding |
| lg    | 24px  | Card padding    |
| xl    | 32px  | Page margins    |

## Elevation

| Level | Usage           |
| ----- | --------------- |
| 0     | Flat surfaces   |
| 2     | Cards, panels   |
| 4     | Dropdowns       |
| 8     | Hover states    |
| 16    | Dialogs, modals |
| 24    | Tooltips        |

## Border Radius

| Token | Value | Usage           |

| md    | 8px   | Cards           |
| lg    | 16px  | Large cards     |
| full  | 50%   | Avatars, badges |
