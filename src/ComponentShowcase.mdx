## Feature Cards

The main action cards on the homepage. Used for primary user actions.

**Component:** `FeatureCard`
**Location:** [src/components/FeatureCard.js](src/components/FeatureCard.js)

```jsx
import FeatureCard from "./components/FeatureCard";
import BuildIcon from "@mui/icons-material/Build";

<FeatureCard
  title="BUILD NEW RESUME"
  icon={<BuildIcon sx={{ fontSize: 40 }} />}
  tooltip="Start building a new resume from scratch"
  color="#9d99e5"
  onClick={() => handleBuild()}
  disabled={false}
/>;
```

**Props:**
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | string | required | Card label text |
| `icon` | ReactNode | required | MUI icon component |
| `tooltip` | string | required | Hover tooltip text |
| `color` | string | `#9d99e5` | Accent color for hover |
| `onClick` | function | - | Click handler |
| `disabled` | boolean | `false` | Disable interactions |

## Text Input

Multi-line text input for job descriptions and personal stories.

**Component:** `TextInput`
**Location:** [src/components/TextInput.js](src/components/TextInput.js)

```jsx
import TextInput from "./components/TextInput";

<TextInput
  title="Job Description"
  placeholder="Paste the job posting here..."
  value={jobDescription}
  onChange={(e) => setJobDescription(e.target.value)}
/>;
```
